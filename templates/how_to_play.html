Principes de base

Le Jeu Des Images est conçu pour vous aider à mémoriser les visages et les noms de vos collègues de manière ludique et interactive.

Chaque mode de jeu propose un défi différent, mais l'objectif reste le même : associer correctement les visages aux noms, équipes et postes.

Modes de jeu classiques

Mode Normal

Dans ce mode, vous voyez la photo d'un collaborateur et devez identifier son entreprise, son équipe, son nom et son poste. Vous gagnez des points pour chaque bonne réponse.

Mode Inversé

À l'inverse du mode normal, vous voyez un nom et devez identifier la personne parmi plusieurs photos. Ce mode teste votre mémoire visuelle.

Mode Pixelation

L'image d'un collaborateur apparaît progressivement, devenant de plus en plus nette. Plus vite vous identifiez la personne, plus vous gagnez de points.

Mode Chronométré

Similaire au mode normal, mais avec une limite de temps pour répondre. La pression est à son comble !

Modes de jeu expérimentaux

Découvrez nos modes de jeu expérimentaux pour des défis encore plus variés :

  • Mode Mémoire : Retrouvez les paires correspondantes, comme dans un jeu de memory.
  • Mode Équipe : Identifiez les membres d'une équipe spécifique.
  • Mode Indice : Utilisez des indices pour identifier la personne.
  • Et bien d'autres à découvrir par vous-même !

Astuces pour progresser

  • Commencez par le mode normal pour vous familiariser avec les visages et les noms.
  • Utilisez le mode pixelation pour tester votre capacité à reconnaître rapidement vos collègues.
  • Jouez régulièrement pour améliorer votre mémoire des visages et des noms.
  • N'hésitez pas à explorer tous les modes de jeu pour une expérience complète.
